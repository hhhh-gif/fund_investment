# 投资监控系统

一个基于 Flask + ECharts 的实时指数与基金监控系统，支持自动刷新、分类统计、智能投资建议等功能。

## 功能特性

- 📈 **实时监控**：支持上证指数、深证成指、创业板指等指数，以及自定义基金代码的实时涨跌幅监控
- 📊 **数据可视化**：使用 ECharts 绘制实时走势图表，支持增量更新
- 📉 **市场概览**：分类统计上涨 / 下跌 / 平盘标的（指数 / 基金分别统计），显示最大涨幅 / 最大跌幅
- 💡 **智能投资建议**：根据市场整体走势生成投资策略建议
- ⚙️ **灵活配置**：支持自定义监控标的和刷新间隔

## 快速开始

### 1. 环境依赖

```
pip install flask flask-cors requests
```

### 2. 启动项目

```
python app.py
```

访问 http://127.0.0.1:8080 即可进入配置中心，配置完成后前往实时监控页面查看数据。

## 使用说明

1. **配置监控标的**

   - 在配置中心页面，按格式填写需要监控的指数（名称 | 代码，每行一个）和基金代码（多个用空格分隔）
   - 设置刷新间隔（建议 ≥ 30 秒）
   - 点击「保存配置」

2. **查看实时监控**

   - 点击「前往实时监控」，即可查看指数和基金的实时涨跌幅、走势图表和市场概览
   - 页面会根据配置的间隔自动刷新数据

3. **投资建议**
